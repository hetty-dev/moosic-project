 Moosic Project

 Project Overview
This project analyzes music listening data to provide insights into user preferences and generate music recommendations.

 Objectives
- Analyze music listening patterns
- Identify popular genres and tracks
- Build a simple recommendation system based on user behavior

 Tools & Skills
- Python
- pandas, numpy
- matplotlib, seaborn
- Basic machine learning concepts

 Key Insights
- Certain genres dominate user listening habits
- User behavior can help predict music preferences
- Recommendations improve engagement with less popular tracks
